Python 3.9.7

Clone repository
```cmd
git clone https://github.com/EricZoop/mqttcasino.git
cd mqttcasino
```

Create enviorment
```cmd
python -m venv venv
```

Activate
```cmd
venv\Scripts\activate
```

Install dependencies
```cmd
pip install -r requirements.txt
```

Run
```cmd
python app.py
```
